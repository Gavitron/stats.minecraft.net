Snooped Minecraft Stats

When we asked the community in early 2012 regarding snooping non-personal data about
Minecraft players (more info here) we wanted to share that data back to you.

This place is the home of that data, and it can be downloaded below.
If you do something cool with the data sets, tweet with #mcstats so we can have a peek.
